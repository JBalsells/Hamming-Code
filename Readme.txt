1. Ejecute el programa Hamming.py (Compiled python)
2. Ingrese un valor deseado de numero de bits "K"
	en la pantalla le aparecerán las matrices de:
		a)Paridad
		b)Identidades
		c)Generadora
		d)Chequeo de paridad
		
	aparecerá el codigo posible a transmitir con una numeración 
	al lado izquierdo, con esta numeración se elige el código a
	enviar.

3. aparecerá un mensaje que dice "ingrese mensaje a transmitir"
	en este lugar ingresamos un numero de Identificacion del
	codigo a transmitir mostrado anteriormente.
4. aparecerá un mensaje que dice "ingrese posicion de error de mensaje"
	en este lugar indicamos el lugar del mensaje, por ejemplo si
	agregamos un "2" en 6 bits se agregará de la siguiente manera:
	"0 0 1 0 0 0 0"
5. se mostrarà el mensaje enviado junto a la matriz de chequeo de paridad.
